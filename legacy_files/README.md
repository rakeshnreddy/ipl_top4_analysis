# Legacy Files

This directory holds local scratch files that were present in the working tree
but are not part of the active IPL Playoff Pulse static app or data pipeline.

The current implementation uses `extract_table.py` and the React frontend under
`frontend/ipl-analyzer-frontend/`. Generated deploy data lives in
`frontend/ipl-analyzer-frontend/public/data/ipl-2026.json`, with legacy JSON
compatibility files generated from the same payload.
